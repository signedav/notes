Kapitel 4
Dokumentation erstellen

Wenn Sie den Menu-Punkt Berichte/Objektkatalog... ([8], Kap. 3.1.6) wählen (Abb. 4.1), müssen Sie einen Datenamen wählen und dann wird eine HTML-Datei erzeugt.


Abbildung 4.1: Menu-Punkt Berichte/Objektkatalog....

In dieser Datei werden alle Modellelemente zusammen mit der Beschreibung aufgelistet. Sie können die Datei mit Hilfe Ihres Web-Browser ansehen und ausdrucken (Abb. 4.2).
