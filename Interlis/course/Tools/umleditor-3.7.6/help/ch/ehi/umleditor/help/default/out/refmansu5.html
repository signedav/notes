4.2.3 INTERLIS 2-Datei

Eine INTERLIS 2-Datei siehe Abb. 4.10 entspricht einem Paket (engl. Package) nach UML.

Für weitere Angaben zu Regeln und Eigenschaften siehe INTERLIS 2 - Referenzhandbuch 2.3.


Abbildung 4.10: Dialog - INTERLIS 2-Datei

FELD

BESCHREIBUNG

Tabelle

Listet Sprache und zugehörigen Dateinamen auf. Ein Modell-Element INTERLIS 2-Datei repräsentiert eine INTERLIS-Modelldatei (siehe Kap. D.4). Wenn das Feld Dateiname ein relativer Dateipfad ist, dann ist er relativ zum Verzeichnis, in dem die UML-Editor-Datei gespeichert ist. Mit der Popup-Menu-Funktion Ändern... lässt sich ein Tabelleneintrag ändern.

Sprachversion

Hinweis auf die Version der verwendeten INTERLIS-Spezifikation. Diese Version des Editors unterstützt die Version 2.2.
