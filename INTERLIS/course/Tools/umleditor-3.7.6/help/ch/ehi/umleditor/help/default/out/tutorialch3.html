Kapitel 3
Daten modellieren

Im Folgenden wird ein Ausschnitt aus dem Datenmodell Roads (aus dem INTERLIS-Referenzhandbuch, Anhang C) modelliert.

Nachdem Sie den UML-Editor gestartet haben, erscheint das Hauptfenster (Abb. 3.1).


Abbildung 3.1: Hauptfenster des UML-Editors.

  • Der Navigationsbereich auf der linken Seite des Hauptfensters des UML-Editor bietet eine schnelle Möglichkeit zu einem bestimmten Element des Modells zu gelangen.
  • Im Modellierbereich können Sie auf einzelnen Diagrammen grafisch Daten modellieren.
  • Im Dokumentationsbereich können Sie zum aktuell ausgewählten Modellelement eine Beschreibung eingeben.

Weitere Informationen finden Sie im Referenzhandbuch ([8], Kap. 3).
